# auto_notary
this program just try to auto notary data cap request by connect google form and filecoin.
it is intend to use on non mainnet

## Run
```
NAME:
   auto-notary run - Start auto notary

USAGE:
   auto-notary run [command options] [arguments...]

OPTIONS:
   --credFile value
   --from value
   --spreadsheetID value
   ```
